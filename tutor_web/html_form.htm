一般的 HTML 標籤，是為了呈現資訊，讓使用者瀏覽；但是透過表單，可以讓使用者與伺服器進行互動。由於本篇內容仍處於 HTML 的階段，所以這裡會先介紹表單的外觀。有關於如何傳送資料讓伺服器端的方法，主要是藉由相關的屬性，負責指定送出方法和送達對象等等，細節會在往後介紹。

表單的輸入框和按鈕，主要透過 input 標籤的 type 屬性來變化，例如 type 設為 reset 時，會呈現出一個按鈕，按下後可將表單內容恢復為預設值；而 value 屬性，則用來指定輸入框的預設內容或按鈕的顯示文字等；name 屬性則主要讓表單的接收方知道是哪個按鈕或輸入框的資料。另外，type 設為 image 時，則可用圖片製作送出按鈕(此處未展示)；而若利用 button 標籤，則可製作出任意的按鈕。以下是表單當中，關於輸入框和按鈕的範例：

﻿<html>
<head>
	<title>標題在這裡</title>
</head>
<body bgcolor="#ffffcc">

<form>
	姓名：<input type="text" name="a" value="王小龜" size="20" maxlength="10"><br><br>
	密碼：<input type="password" name="b" size="20" maxlength="10"><br><br>
	內容：<br><textarea rows="10" cols="20">請在此輸入內容</textarea></br><br>
	<button type="button" name="c"><img src="../pics/alan.jpg" width="50"><br>來戳我~</button>
	<input type="button" name="d" value="這是一般按鈕"><br>
	<input type="submit" name="e" value="這是送出按鈕">
	<input type="reset"  name="f" value="這是清除按鈕">
</form>

</body>
</html>

再來是關於選單的範例。其中，為了讓往後將表單傳送至伺服器時，讓伺服器端知道是哪個項目被選取，所以會需要 value 屬性。label 標籤則可讓使用者在點選到該文字而非選項鈕本身時，也可以進行選擇，不過要注意 id 必須正確對應，否則會產生錯亂的效果。另外，還可以透過 optgroup 標籤，將下拉式選單(select)的選項進行群組化；而若將下拉式選單加上 size 屬性，則可顯示成"列示方塊"。 

﻿<html>
<head>
	<title>標題在這裡</title>
</head>
<body bgcolor="#ffffcc">

<form>
	你覺得如何：
	<input type="radio" name="test" value="選項a" id="a"><label for="a">這是選項a</label>
	<input type="radio" name="test" value="選項b" id="b" checked><label for="b">這是選項b</label>
	<br><br>
	
	單選方塊是靠 name 屬性來分組：
	<input type="radio" name="test2" value="選項c" id="c"><label for="c">這是選項c</label>
	<input type="radio" name="test2" value="選項d" id="d" checked><label for="d">這是選項d</label>
	<br><br>
	
	多選方塊：
	<input type="checkbox" name="test3" value="x" id="x" ><label for="x">這是選項x</label>
	<input type="checkbox" name="test3" value="y" id="y" checked><label for="y">這是選項y</label>
	<input type="checkbox" name="test3" value="z" id="z" checked><label for="z">這是選項z</label>
	<br><br>
	
	下拉式選單：
	<select>
		<option value="16">16~20</option>
		<option value="21">21~25</option>
		<option value="26" selected>26~30</option>
		<option value="31">31~35</option>
		<option value="31">36~40</option>
		<option value="31">41~45</option>
	</select>
	<br><br>

	列式方塊：
	<select size=3>
		<option value="16">16~20</option>
		<option value="21">21~25</option>
		<option value="26" selected>26~30</option>
		<option value="31">31~35</option>
		<option value="31">36~40</option>
		<option value="31">41~45</option>
	</select>
</form>

</body>
</html>

在 HTML 5 當中，新增了一些 type 的性質與屬性，以下介紹其中一些：

﻿<!DOCTYPE html>
<html>
<head>
	<title>標題在這裡</title>
</head>

<body bgcolor="#ffffcc">

<form>
	請選擇數字：
	<input type="range" min="0" max="10" step="0.5" value="2.5">
	<br><br>
	
	請選擇顏色：
	<input type="color">
	<br><br>
	
	請選擇日期：
	<input type="date" min="2013-02-19" max="2013-06-18" step="7">
	<br><br>
	
	進度條：
	<progress value="20" max="100"></progress>
	<br><br>
	
	e-mail：
	<input type="email" required multiple>
	<br><br>
	
	近來可好？：
	<input type="text" placeholder="近來可好？">
	<br><br>
	
	禁止自動完成：
	<input type="text" autocomplete="off">
	<br><br>
	
	list 屬性與 datalist：
	<input type="text" placeholder="近來可好？" list="isGood">
	<datalist id="isGood">
		<option value="好" label=":)">
		<option value="普通" label=":|">
		<option value="不好" label=":(">
	</datalist>
	<br><br>
	
	<input type="submit">
</form>

</body>
</html>

本篇內容將介紹如何安裝 Laravel Framework。 以下的說明以 Windows 7 為例，並假設已安裝過 XAMPP。

Laravel 使用 Composer 這個工具來管理 PHP 的依賴性，因此，我們首先要安裝 Composer。 Windows 的使用者，可以到 https://getcomposer.org/download/ 下載 Windows 的安裝程式來進行。 開始安裝的提示畫面如下：

詢問是否需要滑鼠右鍵選單開啟的功能，此處以預設的"否"進行即可：

指定 php.exe 的路徑，如果你是安裝 XAMPP 組合包，那應該會與圖中相同：

安裝設定確認：

安裝時會從網路下載一些檔案，此為下載結束後的畫面， 內容是告訴你環境變數 PATH 已經設定，不過目前開啟的視窗可能不會載入變更， 也許會需要開啟新的命令視窗，甚至重新登入：

Compoer 安裝完成：

接下來要開始安裝 Laravel，請開啟命令提示字元視窗， 輸入「composer global require "laravel/installer=~1.1"」並等待完成。 此為完成後的畫面：

接下來，請把"C:\Users\geniusturtle\AppData\Roaming\Composer\vendor\bin"加到 Path 環境變數當中。 (請記得換成自己的帳號！如果你自訂了安裝路徑也記得要更換目錄。)

我們現在可以開始建立一個新專案。 你可以找一個想要放置專案的目錄(此例為D:\)， 開啟命令提示字元，輸入「laravel new 專案名稱」來進行， 這個指令會幫你在該目錄下建立一個名稱是剛才訂立的專案名稱的資料夾， 裡面會包含 Laravel Framework 的基本架構。 下圖是完成後的畫面，專案名稱使用 laravelTest：

目錄建立完成後，請在 Apache 的設定檔(httpd.conf)裡面， 加入一個指向"專案名稱\public"的資料夾，例如以下： 

Alias /laravelTest "D:\laravelTest\public"
<Directory "D:\laravelTest\public">
    Options Indexes MultiViews
    AllowOverride All
    Require all granted
</Directory>

若是設定正確，則以上述方式為例時，在瀏覽器網址列輸入"http://localhost/laravelTest"， 就會看到以下畫面(如果出現 403 forbidden， 請在 public 資料夾當中的 .htaccess 加入"Options +FollowSymLinks")：

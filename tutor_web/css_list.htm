以下一步步地說明如何做出類似的選單。

首先，這是選單的基礎架構：

﻿<html>
<head>
</head>
<body bgcolor="#ccccff">

<ul class="menu">
	<li>本系介紹
		<ul>
			<li>一二三四</li>
			<li>五六七八</li>

	</li>

滑鼠經過時，將下一層選單呈現：

﻿<html>

<body bgcolor="#ccccff">

		</ul>
	</li>

</body></html>

接著，我們要修正下層選單展開時，上層選單會移動的問題。因此，可以把 li 的定位設定為 relative，並使其包住的下層 ul 在呈現時為 absolute，以讓第二層選單出現時，不會影響到第一層選單的動作(提示：各位可以嘗試看看，為什麼第一層用 relative 而不用 absolute)：

﻿<html>
<head>
	<style>
		.menu li {
			position: relative;
		}
		.menu li ul {display: none}
		.menu li:hover > ul {
			display:block;
			position: absolute;
		}
	</style>
</head>
<body bgcolor="#ccccff">

<ul class="menu">
	<li>本系介紹
		<ul>
			<li>一二三四</li>
			<li>五六七八</li>
		</ul>
	</li>
	<li>系上成員
		<ul>
			<li>教授成員</li>
			<li>行政人員</li>
		</ul>
	</li>
	<li>研究領域
		<ul>
			<li>實驗室</li>
			<li>影音網</li>
		</ul>
	</li>
</ul>

</body></html>

接著，把位置做一些調整，以及一些後續處理。並且，多階層的選單也可以依此類推(提示：為什麼要自行指定寬度和邊距？可以利用 Chrome 的開發者工具快速的觀察看看)：

﻿<html>
<head>
	<style>
		.menu li {
			position: relative;
			width: 70px;
			list-style:none;
		}
		.menu li ul {display: none}
		.menu li:hover > ul {
			display:block;
			position: absolute;
			left: 100%;
			top: 0;
			width: 70px;
			padding:0;
		}
	</style>
</head>
<body bgcolor="#ccccff">

<ul class="menu">
	<li>本系介紹
		<ul>
			<li>一二三四</li>
			<li>五六七八</li>
		</ul>
	</li>
	<li>系上成員
		<ul>
			<li>教授成員
				<ul>
					<li>第一位</li>
					<li>第二位</li>
					<li>第三位</li>
				</ul>
			</li>
			<li>行政人員</li>
		</ul>
	</li>
	<li>研究領域
		<ul>
			<li>實驗室
				<ul>
					<li>第一間</li>
					<li>第二間</li>
					<li>第三間</li>
				</ul>
			</li>
			<li>影音網</li>
		</ul>
	</li>
</ul>

</body></html>

如果希望第一層選單是橫的，則需要利用 float:left，並且記得將第二層選單設計為向下展開，應該就可以達成效果。

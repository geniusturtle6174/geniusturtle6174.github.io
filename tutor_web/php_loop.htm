PHP 的 for 迴圈語法，是用分號將變數初始值、迴圈繼續的條件，以及變數遞增值用分號區隔；要重複執行的部份，必須用大括號括住。請試著回答，下面範例會輸出什麼內容：

﻿<html><head></head><body bgcolor="#ccccff">

<?php

$x = 5;

for( $i = 1; $i <= $x; $i = $i + 1){
	echo "$i<br>";
}

?>

</body></html>

PHP 的 while 迴圈，則跟大多數語言一樣，把允許繼續執行的條件，放在小括號中（如果你學過的是 Python，則請記得加上小括號） ；要重複執行的部份，則一樣必須用大括號括住。請試著回答，下面範例會輸出什麼內容：

﻿<html><head></head><body bgcolor="#ccccff">

<?php

$x = 5;
$i = 1;

while( $i <= $x ){
	echo "$i<br>";
	$i = $i + 1;
}

?>

</body></html>

你當然也可以撰寫巢狀迴圈，也就是迴圈裡面包著另一個迴圈。但需要注意的是，內層迴圈用來跳動的變數，不能跟外層迴圈一樣，否則會產生錯誤：

﻿<html><head></head><body bgcolor="#ccccff">

<p>正確的使用：<br>
<?php

for( $i = 1; $i <= 2; $i = $i + 1){
	for( $j = 1; $j <= 3; $j = $j + 1){
		echo "$i<br>";
	}
}
?>
</p>

<p>錯誤的使用：<br>
<?php

for( $i = 1; $i <= 2; $i = $i + 1){
	for( $i = 1; $i <= 3; $i = $i + 1){
		echo "$i<br>";
	}
}
?>
</p>

</body></html>

如果要中途跳出迴圈，可以使用「break」；如果要直接繼續下一輪迴圈，可以使用「continue」：

﻿<html><head></head><body bgcolor="#ccccff">

<p>break 的範例：<br>
<?php

for( $i = 1; $i <= 5; $i = $i + 1){
	if ( $i == 3){
		break;
	}
	echo "$i<br>";
}
?>
</p>

<p>continue 的範例：<br>
<?php

for( $i = 1; $i <= 5; $i = $i + 1){
	if ( $i == 3){
		continue;
	}
	echo "$i<br>";
}
?>
</p>

<p>巢狀迴圈的範例：<br>
<?php

for( $i = 1; $i <= 2; $i = $i + 1){
	for( $j = 1; $j <= 4; $j = $j + 1){
		if ( $j == 3){
			break;
		}
		echo "$i, $j<br>";
	}
}
?>
</p>

</body></html>

AppServ 是一個在 Windows 平台運行的 WAMP (Windows, Apache, MySQL, PHP)開源工具， 起源於泰國[wiki]。 AppServ 包含了 Apache, PHP, MySQL, phpMyAdmin 等元件，讓任何人都可以輕鬆的架設自己的網站伺服器。

關於 AppServ 的取得，可連線至http://www.appservnetwork.com/， 在首頁直接點選最新的穩定版本下載，本篇使用 2.5.10 版本為範例。 開始安裝後，會先看到歡迎畫面，按 Next：

版權畫面，按 I Agree：

選擇安裝位置。若要改變，可按 Browse 自行選擇，此處使用預設值，按 Next：



選擇要安裝的元件。此處使用預設值(全部安裝)，按 Next：



輸入伺服器名稱、管理者的 e-mail，以及連接的 port。 伺服器名稱可以自己取一個你喜歡的； port 維持預設即可，但若你已經安裝過 IIS 等網頁伺服器，就必須更換。 輸入完後按 Next：



設定 MySQL 資料庫的管理密碼。 另外，也必須設定資料庫的文字編碼，此處建議選擇 UTF-8。 輸入完後按 Install：



安裝完成，畫面上的兩個選項是問你要不要立刻啟動伺服器。 若暫時不想啟動，可以在稍後到「所有程式」→「AppServ」→「Control Server by Service」當中，點選「ApacheStart」來進行。 按 Finish：



若在瀏覽器網址列輸入 http://127.0.0.1 或者 http://localhost， 可出現以下畫面，則代表安裝成功：



以下這個範例，會利用 php 印出「Hello World!!」， 各位可以把下列程式碼複製下來，存檔命名為「hello.php」，放在「C:\AppServ\www」底下， 此時，如果在網址列輸入「http://localhost/hello.php」，則可以看見與範例連結一樣的結果： 

﻿<html><head></head>
<body bgcolor="#ccccff">

<?php
echo "Hello World!!"
?>

</body></html>

AppServ 預設會將網頁放在「C:\AppServ\www」底下，如果需要放在不同的地方， 則可以打開「C:\AppServ\Apache2.2\conf\httpd.conf」，在最後面加上以下的內容， 並重新啟動 Apache (如果你的 UAC 沒有關閉，則需要用管理員權限重啟)： 

Alias /myTestDir "D:\test"
<Directory "D:\test">
	Options Indexes MultiViews
	AllowOverride None
	order allow,deny
	Allow from all
</Directory>

此時，如果把前述的「Hello World!!」範例，放到「D:\test」底下時， 則在網址列輸入「http://localhost/myTestDir/hello.php」，也會看到同樣的結果。

HTML 中，最常見到的媒體就是文字，因此當然會有許多與文字相關的標籤。以下介紹一些跟文字有關的標籤和其屬性。

寫文章最常見的段落與換行，用法如下：

  • p 標籤代表段落(paragraph)，可以用 align 屬性，指定段落要靠左(left)、置中(center)或靠右(right)。
  • br 標籤代表換行(break)。
﻿<html>
<head>
	<title>標題在這裡</title>
</head>
<body bgcolor="#ffffcc">

<p>段落預設是靠左</p>
<p align="left">當然也可以自己加上靠左的屬性靠左</p>
<p align="center">置中的段落</p>
<p align="right">靠右的段落</p>

第一行<br>
第二行
沒加 br 標籤就不會換到第三行

</body>
</html>

font 標籤可以指定文字的大小、字型、顏色，其中的字型部分，必須要在瀏覽者的電腦裡面也有該字型的狀況下，才會有效。另外， font 標籤在 HTML 5 以後，被建議以 CSS 的設定替代(關於什麼是 CSS，請見下一單元)。 

﻿<html>
<head>
	<title>標題在這裡</title>
</head>
<body bgcolor="#ffffcc">

<font size="1">大小最小是 1</font><br>
<font size="7">大小最大是 7</font><br>
<font size="123456" color="#ff0000">再加上去也不會有用的</font><br>
<font size="7" face="標楷體" color="green">這是標楷體</font>

</body></html>

如果要有粗體(bold)和斜體(italic)的效果，可以使用 b 和 i 標籤：

﻿<html>
<head>
	<title>標題在這裡</title>
</head>
<body bgcolor="#ffffcc">

這是一般體<br>
<b>這是粗體</b><br>
<i>這是斜體</i><br>

</body>
</html>

font 標籤的「face="symbol"」屬性值，會把一般的英文字母顯示成希臘字母，若與上標(sup)和下標(sub)這兩個標籤搭配，就可以產生一些簡單的數學式：

﻿<html>
<head>
	<title>標題在這裡</title>
</head>
<body bgcolor="#ffffcc">

<font face="symbol">ABCDEFGHIJKLMNOPQRSTUVWXYZ</font><br>
<font face="symbol">abcdefghijklmnopqrstuvwxyz</font><br>
y = ax<sup>2</sup> + bx + c<sub>1</sub>

</body>
</html>

經由第一個範例，你知道了換行要靠 br 標籤。但若透過 pre 標籤，則可讓文字的排版以原貌呈現：

﻿<html>
<head>
	<title>標題在這裡</title>
</head>
<body bgcolor="#ffffcc">

<pre>
第一行
第二行
第三行
</pre>

</body>
</html>

如果要顯示特殊文字，例如做為標籤的小於(<)和大於(>)，則可利用「&OO」的形式來顯示，其中的「OO」可以是符號名稱，也可以是字元編號。以下是一些範例，其他更詳細的列表，可以參考 w3schools。

﻿<html>
<head>
	<title>標題在這裡</title>
</head>
<body bgcolor="#ffffcc">

<pre>
&gt;
&lt;
&amp;
&quot;

除了名稱以外，也可以用編號表示：

&#38;
&#62;
&#60;

&#48;
&#65;
&#97;
</pre>

</body>
</html>

之前介紹的 HTML、CSS、JavaScript 等內容，如果只是在自己電腦上觀看，那麼只需要滑鼠在網頁檔案上點兩下，瀏覽器就會自動打開檔案並呈現出來。但是，如果需要讓網頁被別人看見，甚至希望儲存一些資料，以讓不同使用者之間進行互動，則需要自行架設伺服器環境來接受外部連線，或者尋找提供此服務的儲存空間或虛擬主機等等。

這樣子的一個整體環境，除了用來接受連線與送出頁面的網頁伺服器以外，還包含了用來在伺服器端執行運算的程式語言環境、儲存資料的資料庫，以及(選擇性安裝)用來管理資料庫的介面。如果要使用的伺服器端語言是 PHP，則在以上這四項的搭配中，最常見的組合是 Apache + PHP + MySQL (or MariaDB) + phpMyAdmin。以下將分別提及個別安裝以及整包安裝的方法。

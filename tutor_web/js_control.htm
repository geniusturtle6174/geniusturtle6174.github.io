JavaScript 的條件判斷語法與 C 很相似，故以下主要介紹少數「特異功能」。 在 JavaScript 當中，只有數值 0, null 和空字串被視為 false，其餘皆為 true。 

﻿<html>
<head>
</head>
<body bgcolor="#ccccff">

<script>

br2 = "<br/><br/>";

x = 0; (x)?document.write(x + " 為 true"+br2):document.write(x + " 為 false"+br2);

x = null; (x)?document.write(x + " 為 true"+br2):document.write(x + " 為 false"+br2);

x = ""; (x)?document.write(x + " 為 true"+br2):document.write(x + " 為 false"+br2);

x = []; (x)?document.write(x + " 為 true"+br2):document.write(x + " 為 false"+br2);

</script>

</body></html>
字串可以用「==」直接比對，也可以套在 switch 當中： 
﻿<html><head></head>
<body bgcolor="#ccccff">

<script>

week = "三";

switch(week){
	case '一': console.log(week); break;
	case '二': console.log(week); break;
	case '三': console.log(week); break;
	case '四': console.log(week); break;
	case '五': console.log(week); break;
	case '六': console.log(week); break;
	case '日': console.log(week); break;
}

</script>

</body></html>

JavaScript 有一種比較運算「===」，必須要連型別都完全一樣才算是相等： 

﻿<html>
<head>
</head>
<body bgcolor="#ccccff">

<script>

br2 = "<br/><br/>";

document.write(" true == 1 is: " + (true==1) + br2 );

document.write(" true === 1 is: " + (true===1) + br2 );

document.write(" \"123\" == 123 is: " + ("123"==123) + br2 );

document.write(" \"123\" === 123 is: " + ("123"===123) + br2 );

</script>

</body></html>

for 和 while 的用法，也跟 C 相似，但是 JavaScript 提供一種 for-in 迴圈，可以用來列舉物件的性質： 

﻿<html><head></head>
<body bgcolor="#ccccff">

<script>

for(prop in document){
	document.write("document." + prop + " = " + document[prop] + "<br/><br/>");
}

</script>

</body></html>

JavaScript 的條件判斷語法與 C 很相似，基本範例如下：

﻿<html>
<head></head>
<body bgcolor="#ccccff">

<script>

x = prompt("請輸入分數");

if(x >= 60){
	document.write("及格");
}
else {
	document.write("被當");
}

</script>

</body></html>

如果你之前只學過 Python 語言，則需要注意多個條件的寫法。在 JavaScript 裡面沒有「elif」可以使用，而是跟 C 一樣必須使用「else if」，如下：

﻿<html>
<head></head>
<body bgcolor="#ccccff">

<script>

x = prompt("請輸入分數");

if(x >= 90){
	document.write("A+");
}
else if(x >= 80){
	document.write("A 或 A-");
}
else if(x >= 70){
	document.write("B+，B 或 B-");
}
else {
	document.write("剩下的自己算");
}

</script>

</body></html>

其他在 JavaScript 當中存在，但在其他語言裡面不一定有的「特異功能」，則簡單介紹如下面幾個範例。首先，在 JavaScript 當中，只有數值 0, null 和空字串被視為 false，其餘皆為 true。

﻿<html>
<head></head>
<body bgcolor="#ccccff">

<script>

br2 = "<br><br>";

x = 0; document.write("「x = 0」，x 為 " + ((x)?"true":"false") + br2);

x = null; document.write("「x = null」，x 為 " + ((x)?"true":"false") + br2);

x = ""; document.write("「x = \"\"」，x 為 " + ((x)?"true":"false") + br2);

x = []; document.write("「x = []」，x 為 " + ((x)?"true":"false") + br2);

</script>

</body></html>

字串可以用「==」直接比對，也可以套在 switch 當中：

﻿<html>
<head></head>
<body bgcolor="#ccccff">

<script>

week = "三";

switch(week){
	case '一': console.log(week); break;
	case '二': console.log(week); break;
	case '三': console.log(week); break;
	case '四': console.log(week); break;
	case '五': console.log(week); break;
	case '六': console.log(week); break;
	case '日': console.log(week); break;
}

</script>

</body></html>

JavaScript 有一種比較運算「===」，必須要連型別都完全一樣才算是相等：

﻿<html>
<head></head>
<body bgcolor="#ccccff">

<script>

br2 = "<br><br>";

document.write(" true == 1 is: " + (true == 1) + br2 );

document.write(" true === 1 is: " + (true === 1) + br2 );

document.write(" \"123\" == 123 is: " + ("123" == 123) + br2 );

document.write(" \"123\" === 123 is: " + ("123" === 123) + br2 );

</script>

</body></html>

for 和 while 的用法，也跟 C 相似，基本範例如下：

﻿<html>
<head></head>
<body bgcolor="#ccccff">

<script>

document.write("For loop:<br>");
for(i=1; i<=10; i++){
	document.write(i + "<br>");
}

document.write("<br>");

document.write("While loop:<br>");
i = 1;
while(i<=10){
	document.write(i + "<br>");
	i += 1;
}

</script>

</body></html>

JavaScript 還提供一種 for-in 迴圈，跟 Python 的語法類似，只是需要多加一些大小括號，其用途之一是列舉物件的性質：

﻿<html>
<head></head>
<body bgcolor="#ccccff">

<script>

for(prop in document){
	document.write("document." + prop + " = " + document[prop] + "<br><br>");
}

</script>

</body></html>

JavaScript 還提供另外一種 forEach 迴圈，可以讓你很方便的逐一操作陣列的元素如下。此處展示的是其中一個用法，在陣列的篇章中會有其他的範例。

﻿<html>
<head></head>
<body bgcolor="#ccccff">

<script>

arr = [123, 456, 789, 'abc', 'def'];
arr.forEach(item => document.write(item + '<br><br>'));

</script>

</body></html>

CSS 是 Cascading Style Sheet (層疊樣式表)的縮寫，它可以用來讓網頁增加更多表現，使網頁原始碼更簡潔、易於維護。CSS 的基本語法如下：

選擇器 {
	性質一: 值;
	性質二: 值;
	...
}

選擇器代表的是這個樣式要適用的對象，後面用大括號包住你要設定的屬性和值。例如以下範例，可代表將超連結的底線設為沒有，顏色設為藍色：

a {
	text-decoration: none;
	color: blue
}

要在網頁中加入 CSS 設定，有以下幾種方式。

方法一：在 head 的範圍中加入 style 標籤。此方法的 CSS 樣式，會套用到這個頁面的每個被選中的物件(此例中為 a 標籤)。

<head>
	......
	<style type="text/css">
		a {
			text-decoration: none;
			color: blue
		}
	</style>
	......
</head>

方法二：寫在其他檔案(例如 myCss.css)，然後在 html 連結進來。此方法的 CSS 樣式，也會套用到這個頁面的每個被選中的物件。

<head>
	......
	<link rel=stylesheet type="text/css" href="myCss.css">
	......
</head>

其中，myCss.css 這個檔案的內容如下：

a {
	text-decoration: none;
	color: blue
}

方法三：在標籤中使用 style 屬性。此方法的樣式，只對被使用 style 屬性的標籤有效。

<a href="abc.htm" style="text-decoration: none; color: blue">

一般來說，為了讓設計易於維護，採取方法二的外部呼叫，讓內容和樣式分離，是比較好的做法。但本篇以下部分，為了範例的呈現方便，主要將以方法一和三做示範。

HTML 的 meta 標籤，可用於描述該網頁的 metadata。 一般對 metadata 常見的說明，是"描述資料的資料"，亦即資料的屬性。 例如，以一首歌曲來說，它的歌名、作詞(曲)者、所屬專輯、曲風...等等，都算是 metadata。 而以網頁來說，則可以是編碼、關鍵字，甚至是更新頻率等等資訊。

meta 標籤還有其他比較基本的用途，例如若要將網頁每隔五分鐘(300 秒)就重新整理， 則可將這一小段語法加到 head 的範圍當中： 

<meta http-equiv="refresh" content="300">

想要自動轉向到其他網頁，使用方式也非常類似： 
<meta http-equiv="refresh" CONTENT="0;url=test.htm">

meta 標籤的另外一個功能，就是指定網頁的編碼。 如果你的網頁原始檔在編輯時使用的編碼， 跟使用者瀏覽器的預設編碼或自動判定的結果不同時，就會看到亂碼。 而解決這個問題的方法，就是透過 meta 標籤來明確地把網頁該使用的編碼告訴瀏覽器。

語法如下，其中的"UTF-8"就是一種編碼，必須視檔案的實際編碼代換， 例如以正體中文來說，"Big5"就是另一種常用的編碼。 詳盡的列表可以參考 http://www.iana.org/assignments/character-sets/character-sets.xhtml。 

<meta http-equiv="content-type" content="text/html; charset=UTF-8">

這段語法在 HTML 5 裡面比較簡單： 
<meta charset="UTF-8">

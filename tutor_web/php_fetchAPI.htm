Fetch API 跟 Ajax 當中的 XMLHttpRequest 非常相似，但是它提供了更方便的功能和使用方式；唯獨可能需要注意的是，目前(2020/04)為止，仍有大部分的功能屬於實驗性質，並且 IE 不能動。

我們直接來看一個完整的範例，這個範例會顯示伺服器的時間：

<html><head></head>
<body bgcolor="#ccccff">

<p id="showTimeHere"></p>

<script>

function update() {
	fetch('ajax_time.php').then(
		response => response.text()
	).then(
		result => {
			document.getElementById('showTimeHere').innerHTML = result;
			setTimeout(update, 1000);
		}
	).catch(
		err => setTimeout(update, 1000)
	);
}

update();

</script>

</body></html>

php 部分的程式碼則與 Ajax 所需的相同，沒有變化：

<?php
date_default_timezone_set('Asia/Taipei');
echo date('Y-m-d H:i:s'); 
?>

很顯然的，最核心的部分就是要如何來使用 fetch。當 fetch 只代入一個網址作為參數的時候，就會去抓取該網址的內容，如果成功的話就會回傳一個 Promise 的物件(關於此物件的詳細操作，此處暫不介紹)，讓你可以用"then"進行後續的操作。因此，這樣的基本的用法大致如下，其中，在處理物件或錯誤狀況的時候，必須用函式的方式來進行，你可以自由的選用一般函式，或者箭頭函式。

fetch(網址).then(
	處理回傳物件，一般來說是以 text() 或 json() 轉成可用資訊
).then(
	處理可用資訊
).catch(
	處理錯誤狀況
)

Fetch 的預設方法是 GET，如果你想使用 POST 方法送出資料，就需要多填一些參數。我們用以下這組數字相加的範例，來看看 POST 的傳輸方式；若要嘗試 GET 的方式，需要將參數直接帶在網址上，以及切換伺服端的程式碼註解。客戶端程式碼如下：

<html>
<head>
	<meta http-equiv="Content-Type" content="text/html; charset=UTF-8" />
	<script type="text/javascript" src="jquery-1.9.1.min.js"></script>
</head>
<body bgcolor="#ccccff">

<script>
function myCalc(){
	data = new FormData();
	data.append('a', $("#a").val());
	data.append('b', $("#b").val());

	fetch("ajax_ab_post.php", {
		method: "POST",
		body: data
	}).then(
		response => {return response.text();}
	).then(
		result => {document.getElementById('result').innerHTML = result;}
	).catch(
		err => {document.getElementById('result').innerHTML = 'GG';}
	)
}
</script>

<form>
	<input type="text" id="a" value="1"> +
	<input type="text" id="b" value="2">
	<input type="button" value="=" onclick="myCalc();">
	<p id="result"></p>
</form>

</body></html>

伺服端程式碼如下，由於我們在伺服器端依然使用「$_POST」或者「$_GET」來接收，因此客戶端會使用「FormData」或者直接帶在網址上的方式來傳參數。如果各位上網搜尋其他範例，也可以看到前端用 JSON.stringify 傳遞的方式，此時的後端可能就需要用 JSON 相關的方法去解析字串再來做處理。：

<?php
$a = (int)$_POST["a"];
$b = (int)$_POST["b"];
// $a = (int)$_GET["a"];
// $b = (int)$_GET["b"];
echo $a + $b;
?>

XAMPP 在 Mac 上的安裝方式與 Windows 上大致相同，但是多了一種會幫你建立一個虛擬環境的版本。由於後者不方便使用 http://localhost 來連線，因此本篇將說明的是如同其他作業系統一樣，不會建立虛擬環境的版本。請點選首頁上方的"下載"，並找到 Mac 的最新版本，並請再次注意 dmg 檔名的尾巴沒有"vm"字樣。本篇使用 7.3.4 版本為範例。

下載完成後可用一般的方式打開，若出現如圖的視窗，請點選"打開"即可。若有需要輸入密碼的視窗，則請輸入之。

主控台畫面，啟動 Apache 和 MySQL 時，可能會要求你允許該程式的網路連線。之後，就可以用 http://localhost 連線到首頁和 phpmyadmin：

接下來，把自己撰寫的 php 檔案，放在安裝路徑下的 htdocs，應該就可以看到結果。若需要將網頁放在不同的路徑，請打開設定檔，並參考 Windows 篇的設定檔修改方式。若是 Apache 無法啟動，則代表內建的 Apache 正在使用中，請在終端機輸入"sudo apachectl stop"來關掉它。若是 MySQL 也無法啟動，則代表你可能自己安裝了 MySQL，請在終端機輸入"ps -ax | grep mysql"，並將出現的 porcess id 以"kill -9 {pid}"指令關閉

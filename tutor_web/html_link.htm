在說明連結之前，我們先要瞭解「絕對路徑」和「相對路徑」的差異：

  • 相對路徑：相對於現在目錄的路徑表示法。「相對路徑」所指到的檔案或目錄，會隨著現在目錄的不同而改變
  • 絕對路徑：一個絕對的位置，並不會隨著現在目錄的改變而改變

以下是一些舉例說明：

類別範例說明
絕對路徑 file:///C:/Users/acer/Desktop/1639408618.jpg 本機 C 槽上的一個圖片檔案
http://www.cs.nthu.edu.tw 網路上一台 WWW Server 上的一個 HTML 檔案
相對路徑 text.html 或 ./text.html 同一層目錄下的 text.html 檔案
image/text.gif 或 ./image/text.gif image 子目錄下的 text.gif 檔案
../index.html 上一層目錄下的 index.html 檔案
../htm123/abc.html 上一層目錄下 htm123 子目錄的 abc.html 檔案

除非有特殊需求，否則在 HTML 文件中，建議用相對路徑來表示網址。使用相對路徑的好處是：當我們將包含許多網頁的目錄轉移到另一個位置時，雖然絕對位置改變，但是在此目錄下的網頁相對位置並沒有改變，因此我們並不需要大幅修改網頁內的連結。

以下是一個基本的範例：

﻿<html>
<head>
	<title></title>
</head>
<body bgcolor="#ffffcc">

圖片也可以用連結標籤包起來：<br>
<a href="http://mirlab.org"><img src="../pics/safe_turtle.png"></a>

</body>
</html>

可以利用 target 屬性，將連結開啟在不同的視窗：

﻿<html>
<head>
	<title></title>
</head>
<body bgcolor="#ffffcc">

<a href="http://www.ntu.edu.tw" target="abc">台大首頁</a><br>
<a href="http://www.nthu.edu.tw" target="abc">清大首頁</a><br>
<a href="http://www.nctu.edu.tw" target="def">交大首頁</a><br>
<a href="http://www.cs.nthu.edu.tw" target="_blank">清大資工首頁</a><br>

</body>
</html>

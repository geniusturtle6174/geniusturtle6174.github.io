在 HTML 文件中，我們有 ul (無順序的清單)和 ol (有順序的清單)標籤，來呈現清單。以下是 ul 和 ol 標籤的簡單範例：

﻿<html>
<head>
	<title></title>
</head>
<body bgcolor="#ffffcc">

長島冰茶材料：
<ul>
	<li>琴酒 一份</li>
	<li>伏特加 一份</li>
	<li>蘭姆酒 一份</li>
	<li>龍舌蘭 一份</li>
	<li>橙皮酒 一份</li>
	<li>萊姆汁 兩份</li>
	<li>可樂 四份</li>
</ul>

做法：
<ol>
	<li>杯中放入適量冰塊</li>
	<li>將可樂以外的材料依序倒入並攪拌</li>
	<li>將可樂倒入並稍微攪拌</li>
</ol>

</body>
</html>

如果是 ol 標籤，則被其夾住的 li 標籤，可以利用 value 屬性改變編號：

﻿<html>
<head>
	<title></title>
</head>
<body bgcolor="#ffffcc">

<ol>
	<li value="10">...XD</li>
	<li>( ′-`)y-～</li>
	<li>(￣▽￣＃)﹏﹏</li>
	<li value="-2">戳戳</li>
	<li>這是測試</li>
	<li>OAO</li>
	<li>GG</li>
	<li>...QQ</li>
</ol>

</body>
</html>

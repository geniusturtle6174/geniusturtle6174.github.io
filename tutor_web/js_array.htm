JavaScript 的陣列，和 C 和 Python 等大多數語言一樣是從 0 開始索引，但是元素可以比較動態的加入：

﻿<html>
<head>
</head>
<body bgcolor="#ccccff">

<script>

// 大家可能已經在其他語言很熟悉的使用方式
myArr = ['...QQ', 1234, (1==4)];
for(i=0;i<myArr.length;i++){
	document.write("第 " + i + " 元素的內容與型別：" + myArr[i] + ', ' + typeof(myArr[i]) + "<br>");
}

document.write("<hr>")

// 利用 Array 物件
myArr = new Array('...QQ', 1234, (1==4));
for(i=0;i<myArr.length;i++){
	document.write("第 " + i + " 元素的內容與型別：" + myArr[i] + ', ' + typeof(myArr[i]) + "<br>");
}

document.write("<hr>")

// 動態加入
myArr = new Array();
myArr[0] = '...QQ';
myArr[1] = 1234;
myArr[2] = (1==4);
for(i=0;i<myArr.length;i++){
	document.write("第 " + i + " 元素的內容與型別：" + myArr[i] + ', ' + typeof(myArr[i]) + "<br>");
}

document.write("<hr>")

// 元素未定義
myArr = new Array(5);
document.write("myArr 的長度：" + myArr.length + "<br>");
document.write("myArr[0]：" + myArr[0]);

</script>

</body></html>

JavaScript 的 array 有下列方法：

  • concat: 將陣列連接起來
  • join: 把陣列元素串成字串
  • pop: 移除陣列的最後一個元素，並將之傳回
  • push: 加一個新元素到陣列最後面，並傳回新長度
  • reverse: 將陣列反轉
  • shift: 移除陣列的第一個元素，並將之傳回
  • slice: 傳回陣列的其中一段
  • splice: 移除陣列中的元素，並視需要插入新元素。傳回被刪除的元素
  • sort: 排序
  • toString: 傳回陣列的字串表示法
  • unshift: 加一個新元素到陣列最前面，並傳回新陣列

其中的 join，在 JavaScript 當中是「陣列.join(字串)」，亦即 join 方法屬於陣列，但在有些語言例如 Python 當中，是「字串.join(陣列)」，亦即 join 是對於字串的操作。以下示範 join，toString，reverse，以及 sort：

排序時，也可以定義自己的 comparison function 來決定兩個元素的比較結果，其設計原則是當 a 應被視為小於 b 時(a 應排在 b 前面時)，回傳一個負整數； a 與 b 視為相等時回傳 0； a 視為大於 b 時(a 應排在 b 後面時)回傳一個正整數：

﻿<html>
<head>
</head>
<body bgcolor="#ccccff">

<script>

function myCmp(a, b){
	return a.length - b.length;
}

myArr = ["lion", "monkey", "brother", "rhino"];

document.write("字典排序： " + myArr.sort().toString() + "<br><br>");

document.write("使用自定義比較，進行字串長度遞增排序： " + myArr.sort(myCmp).toString() + "<br><br>");

</script>

</body></html>

PHP 的條件判斷，語法和 C 與 JavaScript 大致相同，除了變數名稱前面要加上錢字號以外；如果你先前只學過 Python 或者 BASIC 等語言，則請再記得加上一些必要的小括號以及大括號。下面是一個簡單的範例，請各位試著回答其輸出：

﻿<html><head></head>
<body bgcolor="#ccccff">

<?php
$a = 1;
$b = 2;
$c = 2;

if ($a == $b){
	echo 'Cond 1 matched';
}
else if($b == $c){
	echo 'Cond 2 matched';
}
else {
	echo 'No Cond matched';
}

?>

</body></html>

「或」和「且」的判斷，也跟 C 與 JavaScript 大致相同；如果你先前只學過 Python，請記得以「&&」和「||」來代替 and 和 or。下面也是一個簡單的範例，請各位試著回答其輸出：

﻿<html><head></head>
<body bgcolor="#ccccff">

<?php
$a = 2;
$b = 3;

if ( $a > 0 && $b > 0 ){
	echo 'Matched';
}
else{
	echo 'NOT matched';
}
?>

<br>

<?php
$a = 2;
$b = -3;

if ( $a > 0 || $b > 0 ){
	echo 'Matched';
}
else{
	echo 'NOT matched';
}
?>

</body></html>

常見的錯誤用法也和 C 與 JavaScript 大致相同，例如不能使用一個等號來進行判斷：

﻿<html><head></head>
<body bgcolor="#ccccff">

<?php
$a = 0;

if ( $a = 0 ){
	echo 'Matched';
}
else {
	echo 'NOT matched';
}

?>

<br>

<?php
if ( $a = 3 ){
	echo 'Matched';
}
else {
	echo 'NOT matched';
}

?>

</body></html>

PHP 和 JavaScript 一樣，也有三個等號的用法。兩個等號時會幫你自動轉型後再判斷，而三個等號時則不會幫你轉型，需要型別一樣才算是相等：

﻿<html><head></head>
<body bgcolor="#ccccff">

<?php
$a = 2;
$b = '2';

if ( $a == $b ){
	echo 'Matched';
}
else{
	echo 'NOT matched';
}
?>

<br>

<?php

if ( $a === $b ){
	echo 'Matched';
}
else{
	echo 'NOT matched';
}
?>

</body></html>

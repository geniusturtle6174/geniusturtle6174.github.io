Bootstrap 是一個基於 HTML 與 CSS 的設計模板，它為文字、表單、等元素，設計了一套樣式，讓網頁開發者可以很快速的套用，並讓你能很容易的做到 RWD。它也另外包含了 JavaScript 的對話方塊等功能，但本篇將只對 CSS 的部分進行敘述。

要取得 Bootstrap 有兩種方法，第一種是到官方網站下載已經編譯好和最小化的檔案，第二種是直接取用 CDN (Content Delivery Network，內容傳遞網路)的檔案。在第一種方法中，會提到"編譯"的原因是，Bootstrap 的原始語法是用 Sass (Syntactically Awesome Stylesheets) 寫成，這是一個 CSS 語法的擴充，可以讓你用變數、巢狀結構、迴圈、包含等方法，更靈活且模組化的設計樣式，並需要用相關軟體解析成 CSS 語法；而下載的檔案中的 map 檔案，則是讓你在開啟開發者工具檢視時，很容易的可以找到樣式的原始內容，而不是把非必要排版都移除的最小化版本。若沒有相關的修改需求，則直接使用第二種方法即可，此時只需要用 link 標籤將它引用進來，如下

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css">

接著我們就可以開始使用，但除了將 CSS 檔案引用進來以外，還需要以下設定的配合(不喜歡的同學可以不加有興趣的同學，可以試試看不加入的效果，基本上除了版面比較不美觀以外，影響應該不大)：

  • 加入 HTML5 的 doctype 指示：
    <!DOCTYPE html>

  • 加入適合手機排版的顯示方式，以便在網頁載入時有比較恰當的縮放比例：
    <meta name="viewport" content="width=device-width, initial-scale=1">

  • 用 class 為 container-fluid 或 container 的 div，包住網頁內容。container-fluid 會直接佔滿整個版面，container 則是在固定的視窗大小區間內，版面只會有一種寬度，如果你去翻找 boostrap.css，就會發現以下設定： 
    @media (min-width: 576px) {
  .container {
    max-width: 540px;
  }
}

@media (min-width: 768px) {
  .container {
    max-width: 720px;
  }
}

@media (min-width: 992px) {
  .container {
    max-width: 960px;
  }
}

@media (min-width: 1200px) {
  .container {
    max-width: 1140px;
  }
}

一個最簡單的完整範例如下(再次提醒，本篇不包含 JavaScrtipt，因此不將相關檔案引入)：

﻿<!DOCTYPE html>

如果你的頁面上，需要用很多小方格來排版(例如購物網站的商品展示等)，就很適合使用 grid 來進行。Grid 的用法跟表格很相似，我們需要將內容放在由 row 組成的框框(column)裡面，如下：

﻿<!DOCTYPE html>
<html>
<head>
	<meta name="viewport" content="width=device-width, initial-scale=1">
	<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css">
</head>
<body>

<div class="container-fluid">
	<p>預設是等分：</p>
	<div class="row">
		<div class="col" style="background-color: #eee;">AAA</div>
		<div class="col" style="background-color: #aaa;">BBB</div>
		<div class="col" style="background-color: #eee;">CCC</div>
	</div>

	<p>Bootstrap 的 grid 是分成 12 份，也可以選擇每份的佔比：</p>
	<div class="row">
		<div class="col-9" style="background-color: #eee;">AAA</div>
		<div class="col-3" style="background-color: #aaa;">BBB</div>
	</div>
</div>

</body>
</html>

前面的範例，在視窗較小時會仍會維持原本的等分，如果你希望在視窗較小時從橫的排列切換成直的排列，可以使用「col-sm|md|lg|xl」來指定不同視窗大小時的版面樣貌。其中，四種縮寫分別代表 small, medium, large 與 xlarge，對應的寬度下限分別是 576, 768, 992, 與 1200 像素。例如下面的範例，會在 md 以下的範圍內排成直版(因為每個都佔 100%)，md 到 lg 為一橫版，lg 以上為 2 * 2 的排法：

﻿<!DOCTYPE html>
<html>
<head>
	<meta name="viewport" content="width=device-width, initial-scale=1">
	<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css">
</head>
<body>

<div class="container-fluid">
	<div class="row">
		<div class="col-md-3 col-lg-6" style="background-color: #eee;">AAA</div>
		<div class="col-md-3 col-lg-6" style="background-color: #aaa;">BBB</div>
		<div class="col-md-3 col-lg-6" style="background-color: #eee;">CCC</div>
		<div class="col-md-3 col-lg-6" style="background-color: #aaa;">DDD</div>
	</div>
</div>

</body>
</html>

關於警告訊息(alert)的樣式：

﻿<!DOCTYPE html>
<html>
<head>
	<meta name="viewport" content="width=device-width, initial-scale=1">
	<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css">
</head>
<body>

<div class="container-fluid">
	<p>一個基本用法：</p>
	<div class="alert alert-success">
		Success!
	</div>

	<p>可以在訊息中附上連結：</p>
	<div class="alert alert-info">
		Info! Read <a href="#" class="alert-link">this medssage</a>.
	</div>

	<p>Bootstrap 也有提供可以將訊息關閉的按鈕，但完整的功能需要引入 JavaScript：</p>
	<div class="alert alert-warning">
		Wanging! Read <a href="#" class="alert-link">this medssage</a>.
		<button type="button" class="close" data-dismiss="alert">&times;</button>
	</div>

	<p>其他的樣式們：</p>
	<div class="alert alert-danger">Danger!</div>
	<div class="alert alert-primary">Primary!</div>
	<div class="alert alert-secondary">Secondary!</div>
	<div class="alert alert-dark">Dark!</div>
	<div class="alert alert-light">Light!</div>
</div>

</body>
</html>

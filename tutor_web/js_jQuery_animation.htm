需要注意的是，jQuery 只能針對"單一數值"的屬性進行動畫，如果是背景色等包含多個數值者，需要另外的函式庫才能做動畫變更。

jQuery 的動畫呼叫並不會阻斷程式的執行：

﻿<html>
<head>
	<script type="text/javascript" src="jquery-1.9.1.min.js"></script>
</head>
<body bgcolor="#ccccff">

<div id="ball" style="position:relative;width:50px;height:50px;background:#000;border:1px solid;border-radius:100%"></div>
<p id="myDisplay"></p>

<script>
$("<span>OAO " + Date().toString() + "</span><br/>").appendTo("#myDisplay");
$("#ball").animate({left: '+=70%'}, 5000);
$("<span>QQ " + Date().toString() + "</span><br/>").appendTo("#myDisplay");
</script>

</body></html>

但是若有多個動畫呼叫，則會排入佇列(queue)當中執行：

﻿<html>
<head>
	<script type="text/javascript" src="jquery-1.9.1.min.js"></script>
</head>
<body bgcolor="#ccccff">

<div id="ball" style="position:relative;width:50px;height:50px;background:#000;border:1px solid;border-radius:100%"></div>

<script>
$("<span>1 " + Date().toString() + "</span><br/>").appendTo("#myDisplay");
$("#ball").animate({left: '+=70%'},1000);
$("<span>2 " + Date().toString() + "</span><br/>").appendTo("#myDisplay");
$("#ball").animate({top: '+=70%'},1000);
$("<span>3 " + Date().toString() + "</span><br/>").appendTo("#myDisplay");
$("#ball").animate({left: '-=70%'},1000);
$("<span>4 " + Date().toString() + "</span><br/>").appendTo("#myDisplay");
$("#ball").animate({top: '-=70%'},1000);
$("<span>5 " + Date().toString() + "</span><br/>").appendTo("#myDisplay");
</script>

</body></html>

jQuery 替動畫維護的佇列名稱是 fx，我們也可以自己將函式排進佇列執行：

﻿<html>
<head>
	<script type="text/javascript" src="jquery-1.9.1.min.js"></script>
</head>
<body bgcolor="#ccccff">

<input type="button" id="go" value="GO~~">
<div id="ball" style="position:relative;width:50px;height:50px;background:#000;border:1px solid;border-radius:100%"></div>
<p id="myDisplay"></p>

<script>
$("#go").queue('myQueue', function(){
	$("<span>1 " + Date().toString() + "</span><br/>").appendTo("#myDisplay");
}).queue('myQueue', function(){
	$("#ball").animate({left: '+=70%'},1000);
}).queue('myQueue', function(){
	$("<span>2 " + Date().toString() + "</span><br/>").appendTo("#myDisplay");
}).queue('myQueue', function(){
	$("#ball").animate({top: '+=70%'},1000);
}).queue('myQueue', function(){
	$("<span>3 " + Date().toString() + "</span><br/>").appendTo("#myDisplay");
});

$("#go").click(function(){ $("#go").dequeue('myQueue') });
</script>

</body></html>

而如果要在 Linux 下架設網頁伺服器，則通常會將 Apache, PHP, MySQL, phpMyAdmin 這四樣元件分開安裝。以下是以 Ubuntu Linux 為例，使用 apt-get 安裝相關套件以及細部設定的指令，你必須有 root 權限才能執行相關指令。以下是安裝部分之指令與說明，為了醒目，指令部分將以引號夾住：

  • 安裝 Apache2: 「sudo apt-get install apache2」
  • 安裝 php: 「sudo apt-get install php5」
  • 安裝 mysql lib for php: 「sudo apt-get install php5-mysql」
  • 安裝 mysql: 「sudo apt-get install mysql-server」
    • 安裝時會詢問帳號密碼
    • 如果是 Server 版，則在安裝整個作業系統時預設會進行 mysql 之安裝
  • 安裝 phpMyAdmin: 「sudo apt-get install phpMyAdmin」
    • 安裝時會詢問帳號密碼

再來是設定部分：

  • 設定 Apace2 相關
    • 修改使用者家目錄下的網頁用資料夾名稱: 「sudo vim /etc/apache2/mods-available/userdir.conf」
      • 該資料夾預設名稱是 public_html
      • 共兩處需修改
    • 讓一般使用者可以利用家目錄下的網頁資料夾: 「sudo a2enmod userdir」
    • 設定完後重新啟動 Apache: 「sudo service apache2 restart」
    • 若要於建立使用者時，自動增加網頁資料夾目錄: 「sudo mkdir /etc/skel/你要的名稱」
      • /etc/skel為新建使用者創造家目錄時之模版目錄
  • 設定 phpMyAdmin 相關
    • 修改 phpMyAdmin 連結路徑: 「sudo vim /etc/apache2/conf.d/phpmyadmin.conf」 (Apache 2.2 版) 或 「sudo vim /etc/apache2/conf-available/phpmyadmin.conf」 (Apache 2.4 版)
      • 在最上面增加一行: Alias /yourNewPath /usr/share/phpmyadmin
      • 修改完後可能需要重啟 Apache

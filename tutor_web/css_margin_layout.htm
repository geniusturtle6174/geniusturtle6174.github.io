邊界(margin)、邊框(border)、邊距(padding)三著的關係如下， 並且，分別又都可以對上下左右四個方向，進行不同的設定：

首先，為了讓視覺效果較為明顯，我們把段落加上邊框 (表格的原始碼比較長，為了教材頁面的縮短，故使用段落進行示範)， 「1px」是指邊框的粗細，「solid」是指邊框的樣式。 「margin:20px」意即指定段落離上下左右邊界的距離皆為 20px (當然，如果內容不夠多，或者視窗縮得不夠小，則下邊界的效果可能不明顯)。 

﻿<html>
<head>
	<style>
		p{border:1px solid; margin:20px;}
	</style>
</head>
<body bgcolor="#ccccff">

<p>這是測試</p>

</body></html>
如果要分別指定上下左右的值，則可分別以 「margin-top」、「margin-bottom」、「margin-left」、「margin-right」來指定。 另一種方法，則是透過給予 margin 屬性不同數目的值，來分別指定：

  • 一個值時：上下左右
  • 兩個值時：上下、左右
  • 三個值時：上、左右、下
  • 四個值時：上、右、下、左

以下示範四個值的指定效果： 
﻿<html>
<head>
	<style>
		p{border:1px solid;margin:20px 100px 2000px 300px;}
	</style>
</head>
<body bgcolor="#ccccff">

<p>這是測試</p>
這裡是段落之外

</body></html>

段落、表格等內容與其邊框的距離，稱為 padding (邊距)。 指定的語法與 margin 非常類似，這裡直接示範四個值的效果： 

﻿<html>
<head>
	<style>
		p{border:1px solid;padding:20px 40px 80px 160px;}
	</style>
</head>
<body bgcolor="#ccccff">

<p>夫難平者，事也。昔先帝敗軍於楚，當此時，曹操拊手，謂天下已定。
──然後先帝東連吳、越，西取巴、蜀，舉兵北征，夏侯授首：此操之失計，而漢事將成也。
──然後吳更違盟，關羽毀敗，秭歸蹉跌，曹丕稱帝：凡事如是，難可逆見。
臣鞠躬盡瘁，死而後已；至於成敗利鈍，非臣之明所能逆睹也。</p>

</body></html>

上下左右的框線粗細，可以分別用 「border-top-width」、「border-bottom-width」、 「border-left-width」、「border-right-width」來指定， 也可以用「border-width」給上不同數目的值來指定： 

﻿<html>
<head>
	<style>
		p{border:solid; border-width:20px 40px 80px 160px;}
	</style>
</head>
<body bgcolor="#ccccff">

<p>這是測試</p>

</body></html>

框線顏色，則可用「border-***-color」或「border-color」來指定： 

﻿<html>
<head>
	<style>
		p{border:solid; border-width:20px;
		  border-color: #f33 #3f3 #33f}
	</style>
</head>
<body bgcolor="#ccccff">

<p>這是測試</p>

</body></html>

框線形式，則是用「border-***-style」或「border-style」來指定。 有以下幾種樣式可以使用：

  • none
  • hidden
  • dotted
  • dashed
  • solid
  • double
  • groove
  • ridge
  • inset
  • outset

其中，預設值為 none，代表沒有邊框。我們首先示範其中幾個： 
﻿<html>
<head>
	<style>
		p{border:#f70 10px;}
	</style>
</head>
<body>

<p style="border-style:double">double</p>
<p style="border-style:dashed">dashed</p>
<p style="border-style:dotted">dotted</p>
<p style="border-style:groove">groove</p>
<p style="border-style:ridge">ridge</p>
<p style="border-style:inset">inset</p>
<p style="border-style:outset">outset</p>

</body></html>

表格中每一儲存格的框線，預設是分開的單線，但是也可以設定為重疊： 

﻿<html>
<head>

		table.test1{border-collapse:separate}
		table.test2{border-collapse:collapse}
	</style>
</head>

</body></html>
border 的 none 和 hidden ，在此處會展現差異。 如果將框線設定為重疊，則 hidden 會最為優先，none 為最後， 其餘樣式若粗細相同時，優先程度由高到低依次為 double, solid, dashed, dotted, ridge, outset, groove, inset

「display:none」和「visibility:hidden」都可以讓元素消失； 不過前者的效果是變成相當於該元素不存在的狀態；而後者是將元素隱形，但仍佔用空間： 

﻿<html><head></head>
<body bgcolor="#ccccff">

何謂四維？一曰禮，二曰義，三曰廉，<span style="display:none">四曰恥</span>。<br/>
禮不踰節，義不自進，廉不蔽惡，<span style="visibility:hidden">恥不從枉</span>。<br/>

</body></html>

在 HTML 標籤中，你會發現有些元素之間會換行，有些則不會， 這是因為它們的 display 屬性不同所致。 「display:block」時代表該元素會獨佔新的一行， 而「display:inline」則會跟在上一個元素後面。 我們先示範一下，如何用 CSS 強制改變其呈現效果： 

﻿<html>
<head>
	<style>
		p.test{display:inline}
		b.test, i.test {display:block}
	</style>
</head>
<body bgcolor="#ccccff">

<p>這是第一段</p>
<p>這是第二段</p>
<i>OAO</i><b>...QQ</b>

<hr>

<p class="test">這是第一段</p>
<p class="test">這是第二段</p>
<i class="test">OAO</i><b class="test">...QQ</b>

</body></html>

當一個元素為「display:inline」時，元素本身(例如圖片、文字)的大小就是它的大小， 所以有一些屬性會無法使用。以下透過對 block 和 inline 元素同時指派一些屬性，來說明其差異： 

﻿<html>
<head>
	<style>
		p{	color: #f33;border: 5px solid #000;
			width: 100px; height: 100px;
			margin-top: 50px; margin-bottom: 50px;
		}
		i{	color: #f33;border: 5px solid #000;
			width: 100px; height: 100px;
			margin-top: 50px; margin-bottom: 50px;
		}
	</style>
</head>
<body bgcolor="#ccccff">

<p>QQ</p>
<i>XD</i>

</body></html>
display 屬性的值，還可以設定成 inline-block 等值， 若有興趣，可直接參考 w3schools。

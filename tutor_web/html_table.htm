要在網頁中顯示表格，可以使用 table 標籤，並利用 tr 來幫表格新增一個 row，以及利用 td 標籤新增一個儲存格。這個範例，會產生一個 2-by-3 的表格，其中，「border="1"」代表表格邊框的寬度是一個像素。 

﻿<html>
<head>
	<title></title>
</head>
<body bgcolor="#ffffcc">

<table border="1">
	<tr>
		<td>1</td>
		<td>2</td>
		<td>3</td>
	</tr>
	<tr>
		<td>4</td>
		<td>5</td>
		<td>6</td>
	</tr>
</table>

</body>
</html>

可以透過下列這些屬性，來對表格的樣子進行調整：

  • width：寬度
  • cellspacing：格子與格子之間的距離
  • cellpadding：資料到格子之間的距離
  • bgcolor：底色
  • align：與段落 (p) 的 align 相同

以下範例，展示了其中幾個：

﻿<html>
<head>
	<title></title>
</head>
<body bgcolor="#ffffcc">

<table border="1" align="center" cellspacing="20" cellpadding="50">
	<tr>
		<td>1</td>
		<td>2</td>
		<td>3</td>
	</tr>
	<tr>
		<td>4</td>
		<td>5</td>
		<td>6</td>
	</tr>
</table>

</body>
</html>

在 td 標籤中，可以透過 rowspan 或 colspan 屬性，來合併儲存格：

﻿<html>
<head>
	<title></title>
</head>
<body bgcolor="#ffffcc">

<table border="1" align="center">
	<tr>
		<td>1</td>
		<td colspan="2">2</td>
	</tr>
	<tr>
		<td>4</td>
		<td rowspan="2">5</td>
		<td>6</td>
	</tr>
	<tr>
		<td>7</td>
		<td>8</td>
	</tr>
</table>

</body>
</html>

﻿<html>
<head>
	<title></title>
</head>
<body bgcolor="#ffffcc">

<table border="1" align="center" cellspacing="20" cellpadding="50">
	<tr>
		<td rowspan="2">1</td>
		<td colspan="2">2</td>
	</tr>
	<tr>
		<td>3</td>
		<td rowspan="2">4</td>
	</tr>
	<tr>
		<td colspan="2">5</td>
	</tr>
</table>

</body>
</html>

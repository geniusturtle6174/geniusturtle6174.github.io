CSS 2 和 CSS 3 當中，定義了一些新的選擇器， 讓你可以依據標籤的屬性甚至於文件結構，來進行元素的選擇。 例如：

  • 「[xxx]」可以選擇屬性為 xxx 的任意元素
  • 「[xxx="yyy"]」可以選擇屬性為 xxx，且值為 yyy 的任意元素
  • 「[xxx~="yyy"]」可以選擇屬性為 xxx，且多個值以空白分隔時，其中有 yyy 的任意元素
﻿<html>
<head>
	<style>
		[xxx]{font-size:30px;}
		[xxx="yyy"]{color:red;}
		[xxx~="ccc"]{color:green;}
		input[xxx="yyy"]{border:10px solid;}
	</style>
</head>
<body bgcolor="#ccccff">

<p xxx="abc">測試一</p>
<p xxx="def">測試二</p>
<p xxx="bbb ccc ddd">測試三</p>
<p xxx="yyy">測試三</p>
<input type="button" xxx="yyy" value="OAO"/>

</body></html>
這個範例的「input[xxx="yyy"]」，代表選擇 input 元素中，屬性與值為「xxx="yyy"」者， 所以你只會在 input 看到格線的改變。

我們也可以選擇出屬性值包含特定字串的元素，例如 CSS3 的新選擇器：

  • [abc^="def"]：屬性 abc，值為 def 開頭
  • [abc$="def"]：屬性 abc，值為 def 結尾
  • [abc*="def"]：屬性 abc，值包含 def

以下利用這些選擇器，示範如何在檔案下載列表中，很方便的加入小圖示： (連結名稱與內容蒐羅自學校各處室與系所網站，本範例不保證其存活) 
﻿<html>
<head>
	<style>
		a[href^="http"]{background: url("../pics/out.gif") no-repeat; padding-left:28px;}
		a[href$="pdf"]{background: url("../pics/pdf.gif") no-repeat; padding-left:28px;}
		a[href$="doc"]{background: url("../pics/doc.gif") no-repeat; padding-left:28px;}
		a[href$="xls"]{background: url("../pics/xls.gif") no-repeat; padding-left:28px;}
	</style>
</head>

<body bgcolor="#ccccff">

CSS 2 的「:before」和「:after」可以用來在某個元素的前面或後面插入內容： 

﻿<html>
<head>
	<style>
		p:before{
			content:"GG";
			color:red;
		}
		p:after{
			content:"QQ";
			color:green;
		}
	</style>
</head>
<body bgcolor="#ccccff">

<p>這是測試</p>

</body></html>
網路上有一些範例，是關於「:before」和「:after」的妙用， 例如 這個範例， 就利用了「:before」和「:after」，再加上 CSS 邊框設定，以及後面會提到的圓角、轉變等方法，畫出一個個精美的小圖示。

前面提到過，要適用於被某元素包住的元素，可以用空白分隔，這個方法不論被包了多少層都適用。 如果只需要適用一層，可以使用「>」來指定。以下展示差別： 

﻿<html>
<head>
	<style>
		ul.AAA li{border:1px solid;}
		ul.BBB > li{border:1px solid;}
	</style>
</head>
<body bgcolor="#ccccff">

<ul class="AAA">
	<li>1
		<ul>
			<li>1-1</li>
			<li>1-2</li>
		</ul>
	</li>
	<li>2</li>
</ul>

<ul class="BBB">
	<li>1
		<ul>
			<li>1-1</li>
			<li>1-2</li>
		</ul>
	</li>
	<li>2</li>
</ul>

</body></html>

"結構化虛擬類別"則可以讓我們更方便的依據 HTML 的內容架構，來選擇出要處理的元素。 例如 CSS2 的「first-child」，以及 CSS3 新增的「last-child」、「nth-child(n)」、 「nth-last-child(n)」等等。 以下示範的是「nth-child(n)」這個選擇器，它的功用是選出該元素的兄弟， 其中的參數 n 可以是：

  • 索引值，從 1 開始：1, 2, 3, ...
  • odd, even
  • 以 n 為變數的公式，如：2n+1

利用這個選擇器，可以很方便地達到表格奇數或偶數列分色的效果。 以下是簡單的範例，示範公式形式的使用： 
﻿<html>
<head>
	<style>
		li:nth-child(4n-1){background-color:#555;}
	</style>
</head>
<body bgcolor="#ccccff">

	<li>...</li><li>...</li><li>...</li><li>...</li><li>...</li><li>...</li><li>...</li><li>...</li>
	<li>...</li><li>...</li><li>...</li><li>...</li><li>...</li><li>...</li><li>...</li><li>...</li>
	<li>...</li><li>...</li><li>...</li><li>...</li><li>...</li><li>...</li><li>...</li><li>...</li>
</ol>

</body></html>

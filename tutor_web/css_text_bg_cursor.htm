font 標籤只能有七種字型大小，但是 CSS 可以很自由的調整字型大小。 另外，在指定色彩時，如果三種顏色的色碼，其兩個位數都個別相同，則可以個別簡寫為一個： 

﻿<html>
<head>
	<style>
		font{font-size:123pt;color:#cc33cc}
		body{font-size:45pt;color:#c3c;}
	</style>
</head>
<body bgcolor="#ccccff">

<font>在font標籤內</font><br/>
在font標籤外

</body></html>
其中，pt 這個單位，在顯示器的預設設定下會等於 1/72 英吋。若要使用像素來當單位，則是 px。

如果要同時應用於多種元素，則可使用逗點分隔； 若是要適用於被某元素包住的元素，則使用空白分隔： 

﻿<html>
<head>
	<style>
		p, h1{color:#7777ff}
		p font{color:#ff7777}
	</style>
</head>

可以利用「id」或「class」屬性，幫標籤加上不同的 id 或類別。 此時，可用以下方法，讓樣式適用於指定的 id 或類別：

  • #ID名：適用於指定ID
  • .類別名：適用於指定類別
  • 元素名#ID名：適用於ID指定之元素
  • 元素名.類別名：適用於類別指定之元素
﻿<html>
<head>
	<style>
		p{color:red} /* 適用於所有段落，不過若指定類別或 ID 則以其為優先 */
		.myClass{color:green} /* 適用於所有 class="myClass" 者 */
		table.myClass{font-size:123px;} /* 只適用於表格中有 class="myClass" 者 */
		#myId{color:blue} /* 適用於 id="myId" 者 */
	</style>
</head>
<body bgcolor="#ccccff">

<p>一般的段落</p>
<p class="myClass">指定class的段落</p>
<p id="myId">指定ID的段落</p>

<table class="myClass"><tr><td>測試</td></tr></table>

</body></html>

「:link」、「:visited」、「:hover」、「:active」這四個選擇器，可以用來改變超連結的樣式。 這四個選擇器在使用時，「:link」和「:visited」必須在最前面，接著依序才是「:hover」和「:active」。 範例如下： 

﻿<html>
<head>
	<style>
		a:link{color:blue}
		a:visited{color:purple}
		a:hover{color:red}
		a:active{color:green}
	</style>
</head>
<body bgcolor="#ccccff">

<a href="http://www.nthu.edu.tw">清華大學首頁</a>

</body></html>

其中的「:hover」和「:active」則也可以用在超連結以外的地方。 另外也有「:focus」，可以設定例如像輸入框等物件被聚焦時的樣式。 例如： 

﻿<html>
<head>
	<style>
		li,input{font-size:123px;}
		li:hover{color:red}
		li:active{color:blue}
		input:focus{color:green}
	</style>
</head>

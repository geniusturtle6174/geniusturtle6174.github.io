font 標籤只能有七種字型大小，但是 CSS 可以很自由的調整字型大小。另外，在指定色彩時，如果三種顏色的色碼，其兩個位數都個別相同，則可以個別簡寫為一個。其中，pt 這個單位，在顯示器的預設設定下會等於 1/72 英吋；若要使用像素來當單位，則是 px；若有興趣了解其他單位，可以參考 CSS Ruler。 

﻿<html>
<head>
	<style>
		font {
			font-size: 123pt;
			color:#cc33cc
		}
		body {
			font-size: 45pt;
			color: #c3c;
		}
	</style>
</head>

在 HTML 標籤中，可以利用「id」或「class」屬性，幫標籤加上不同的 id 或類別。此時，可用以下方法，讓樣式適用於指定的 id 或類別：

  • #ID名：適用於指定 ID
  • .類別名：適用於指定類別
  • 元素名#ID名：適用於 ID 指定之元素
  • 元素名.類別名：適用於類別指定之元素
﻿<html>
<head>
	<style>
		p { /* 適用於所有段落，不過若指定類別或 ID 則以其為優先 */
			color: red
		}
		.myClass { /* 適用於所有 class="myClass" 者 */
			color: green
		}
		#myId { /* 適用於 id="myId" 者 */
			color: blue
		}
		table.myClass { /* 只適用於表格中有 class="myClass" 者 */
			font-size: 123px;
		}
	</style>
</head>
<body bgcolor="#ccccff">

<p class="myClass">指定 class 的段落</p>
<p id="myId">指定 ID 的段落</p>

<table class="myClass"><tr><td>表格裡面的字</td></tr></table>

</body></html>

		a:hover {
			color: red
		}
		a:active {
			color: green
		}

php 當中的變數，命名規則與 C/JavaScript 相同，但是前面必須加上「$」符號。下面是一個簡單的運算範例，但由於 php 程式碼會先經過伺服器解析執行，所以我們在網頁的原始碼只會看到 3 這個結果：

﻿<html><head></head>
<body bgcolor="#ccccff">

<?php
$a = 1;
$b = 2;
echo $a + $b;
?>

</body></html>

與 JavaScript 的 src 相似，php 可以利用「include」或是「require」，將外部檔案含括進來。兩者的差別在於在檔案找不到時，「include」只產生警告，會繼續執行；而「require」會產生錯誤，停止執行。另外，還有「include_once」與「require_once」可以使用，差別只在於加了「_once」，代表只會含括一次。例如，假設有一個檔案名為 myTest.php，其內容如下：

﻿<?php
$a = 1;
$b = 2;
?>

則其他的 php 檔案，可以用「include」來使用其內容：

﻿<html>
<head>
<?php include('myTest.php') ?>
</head>
<body bgcolor="#ccccff">

<?php echo $a+$b; ?>

</body></html>

php 的變數也會自動轉型，你也可以強制進行轉型。若要知道變數類型，則可使用「gettype」：

﻿<html><head></head>
<body bgcolor="#ccccff">

<?php
$a = "123";		echo "a = \"123\", type is: ".gettype($a)."<br>";
$a += 456;		echo "a += 456, type is: ".gettype($a)."<br>";
$a += 78.9;		echo "a += 78.9, type is: ".gettype($a)."<br>";
$a += "10 xyz";	echo "a += \"10 xyz\", type is: ".gettype($a)."<br>"; // This line produces a Notice
$a .= "10 xyz";	echo "a .= \"10 xyz\", type is: ".gettype($a)."<br>";
?>

<hr>

<?php
$b = "123.456";			echo "b = \"123.456\", type is: ".gettype($b)."<br>";
$b = (int)"123.456";	echo "b = (int)\"123.456\", type is: ".gettype($b)."<br>";
$b = (double)"123.456";	echo "b = (double)\"123.456\", type is: ".gettype($b)."<br>";
$b = (string)78.9;		echo "b = (string)78.9, type is: ".gettype($b)."<br>";
?>

</body></html>

字串也可以使用單引號或雙引號，差別在於其中的變數是否會轉換後再輸出：

﻿<html>
<head></head>
<body bgcolor="#ccccff">

<?php
$x = 123;
echo '單引號字串帶變數: $x.<br>';
echo "雙引號字串帶變數: $x.<br>";
?>

</body></html>

陣列也跟 JavaScript 一樣，可以放入不同類型的資料，也可以動態新增。用迴圈處理資料時，除了一般的方法之外，還有「foreach」可以使用：

﻿<html><head></head>
<body bgcolor="#ccccff">

<?php
$test = array(1, 2.3, "4567");
$test[3] = 'abc'; // 動態新增

echo $test[0]." ".gettype($test[0])."<br>";
echo $test[1]." ".gettype($test[1])."<br>";
echo $test[2]." ".gettype($test[2])."<br>";
echo $test[3]." ".gettype($test[3])."<br>";
?>

<hr>

<?php
for( $i = 0 ; $i < count($test) ; $i++ ){
	echo $i."-th element of \$test: ".$test[$i]."<br>";
}
echo '跑完迴圈後的 $i: '.$i;
?>

<hr>

<?php
foreach( $test as $val ){
	echo "Element of \$test: ".$val."<br>";
}
echo '跑完迴圈後的 $val: '.$val;
?>

</body></html>

當然也可以使用 key/value 的形式來指定資料：

﻿<html><head></head>
<body bgcolor="#ccccff">

<?php

$abc = array(123=>456, "789"=>"abc");
$abc['OAO'] = "QQ";

echo $abc[123]." ".gettype($abc[123])."<br>";
echo $abc['789']." ".gettype($abc["789"])."<br>";
echo $abc["OAO"]." ".gettype($abc['OAO'])."<br>";
?>

<hr>

<?php
foreach( $abc as $val ){
	echo "Element of \$abc: ".$val."<br>";
}
?>

<hr>

<?php
foreach( $abc as $key => $val ){
	echo "Element of \$abc[".$key."]: ".$val."<br>";
}
?>

</body></html>

php 在函式內部的變數，與 C 和 JavaScript 有點不一樣。預設情況下，函式內部都是 local 變數，如果要取用 global 變數，則需使用「$GLOBALS」或者「global」：

﻿<html><head></head><body bgcolor="#ccccff">

<?php

$x = 10;
$y = 20;
$z = 30;

function test(){
	$x = 40;
	$GLOBALS['y'] = 50;
	global $z;
	$z = 60;
	echo "in test: ".$x." ".$GLOBALS['y']." ".$z."<br>";
}

echo "Before test: ".$x." ".$y." ".$z."<br>";
test();
echo "After test: ".$x." ".$y." ".$z."<br>";

?>

</body></html>

在表單標籤中，是透過「action」屬性來指定送出的表單要用哪支程式來處理；而「method」屬性則是設定送出的方法，分為：

  • get: 會附帶在 url 裡面送出，長度較為受限
  • post: 嵌在封包內部送出，可允許較大量的資料

而用「name」屬性，即可讓 php 取出送過來的值。此時需要留意，雖然 html 不分大小寫，但是 php 跟 JavaScript 都一樣會區分。以下是「get」的範例，：

﻿<html><head></head>
<body bgcolor="#ccccff">

<?php
if(isset($_GET['OAO']) && isset($_GET['QQ'])){
	echo $_GET['OAO'] + $_GET['QQ'];
}
?>

<form action="php_formGet.php" method="get">
	<input type="text" name="OAO">
	<input type="text" name="QQ">
	<input type="submit">
</form>

</body></html>

「post」的範例：

﻿<html><head></head>
<body bgcolor="#ccccff">

<?php
if(isset($_POST['OAO']) && isset($_POST['QQ'])){
	echo $_POST['OAO'] + $_POST['QQ'];
}
?>

<form action="php_formPost.php" method="post">
	<input type="text" name="OAO">
	<input type="text" name="QQ">
	<input type="submit">
</form>

</body></html>

上述範例的「isset」，可以判斷變數是否存在，以使得一開始不會印出訊息。不過「isset」只會在變數未宣告、未定義或者值為 null 時回傳 false，如果需要判斷是否為空字串，必須使用「==""」或者「empty」進行。

與按鈕互動：

﻿<html><head><style>body, select{font-size:50px}</style></head>
<body bgcolor="#ccccff">

<?php
for($i=1;$i<=10;$i++){
	echo "按鈕 $i 是否被按下: ".(isset($_POST[$i])?"是":"否")."<br/>";
}
?>

<form action="php_formBtn.php" method="post">
	<?php
		for($i=1;$i<=10;$i++){
			echo "<input type=\"submit\" name=\"$i\">";
		}
	?>
</form>

</body></html>

若要與選單互動，則需在名稱後面加上「[]」，以代表形成陣列(各位可以自行測試看看不加「[]」的效果)：

﻿<html><head><style>body, select{font-size:70px}</style></head>
<body bgcolor="#ccccff">

<?php
if(isset($_POST['number']) && gettype($_POST['number']) == 'array'){
	$sum = 0;
	foreach($_POST['number'] as $i){
		$sum += $i;
	}
	echo $sum;
}
?>

<form action="php_formSelect.php" method="post">
	<select name="number[]" multiple>
		<?php
			for($i=1;$i<=10;$i++){
				echo "<option>".$i."</option>";
			}
		?>
	</select>
	<input type="submit">
</form>

</body></html>

與多選方塊互動，也要加上「[]」：

﻿<html><head><style>body, select{font-size:70px}</style></head>
<body bgcolor="#ccccff">

<?php
if(isset($_POST['number']) && gettype($_POST['number']) == 'array'){
	$sum = 0;
	foreach($_POST['number'] as $i){
		$sum += $i;
	}
	echo $sum;
}
?>

<form action="php_formChkbox.php" method="post">
	<?php
		for($i=1;$i<=10;$i++){
			echo "<input type=\"checkbox\" name=\"number[]\" id=\"$i\" value=\"$i\">";
			echo "<label for=\"$i\">$i</label>&nbsp;";
		}
	?>
	<input type="submit">
</form>

</body></html>

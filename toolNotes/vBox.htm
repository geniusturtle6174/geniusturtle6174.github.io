- 設定主客端共享資料夾之方式
  - 於 VirtualBox 主視窗先設定共享資料夾與名稱
    - 機器 -> 設定值 -> 分享資料夾
    - 加入新的資料夾並命名
  - VirtualBox 客端作業系統視窗設定掛載
    - 裝置 -> 安裝額外客端功能
    - 以下假設閱讀者了解 Linux 操作
    - 以 root 權限執行 VBoxLinuxAdditions.run
    - sudo mount -t vboxsf {主視窗設定分享的名稱} /mnt/{掛載對應位置，需先以root身份建立}

# Course Outline of WEB Programming

## 課程目標

學習網頁設計與架站

## 預先要求

* 基本電腦操作與高中程度之英數能力
* 學過任何一種程式語言

## 相關書目

本課程使用教師自編教材，不指定教科書

## 課程進度

* HTML & CSS (3w)
* JavaScript + HTML5 (5w)
  * 簡介、表單、事件
  * 視窗、預約執行與日期物件、Cookie
  * 通用表示法、jQuery、OpenStreetMap
  * 網頁儲存、拖放操作、畫布、影音
* PHP + MySQL (6w)
  * 環境安裝、簡介、表單
  * SQL、PDO
  * 檔案傳輸、Ajax
  * 範例研究：DVWA
* (Optional) Python 簡介及爬蟲教學(2w)

## 其他說明

* 教學方式
  * 原則為每週約兩小時教師講授 + 約一小時隨堂練習
  * 部分週次得為翻轉教學，課堂時段進行同學作業展示或加強練習
* 以上包含：五到七次作業、一個期末專案、一次期末考試

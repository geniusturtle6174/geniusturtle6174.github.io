# Course Outline of C Programming

## 課程目標

學習 C 語言，以及用程式解決問題的基本能力

## 預先要求

基本電腦操作與高中程度之英數能力

## 相關書目

* 教科用書: 本課程使用教師自編教材，不指定教科書，但建議初學者擁有紙本書籍
* 參考用書: 第一次學 C 就上手、C 程式設計藝術

## 課程進度

* 開發環境介紹/輸入輸出與數值運算(1w)
* 條件判斷/數學函式庫/自訂函式(1w)
* 重複執行/一維陣列(1~1.5w)
* 亂數/遞迴(1w)
* 二維陣列/矩陣運算(1~1.5w)
* 指標(1~1.5w)
* 字串(1w)
* 結構(1w)
* 檔案操作(0.5w)
* 進階主題: 動態規劃(1w)
* 進階主題: 統計學習(1w)
* 延伸主題: Python 程式設計簡介(3w)

* 以上搭配三次考試及 6~8 次作業

本篇以 Windows 10 為例，說明 Python 開發環境的安裝；若使用其他作業系統，請參考網路上的相關資料。關於開發環境的取得，首先可前往 Python 官網 https://www.python.org/，接著將滑鼠滑到「Downloads」上面，再直接點選右邊大顆的按鈕，即可下載最新的版本（教材撰寫時，是 3.12.6）；若你有需要取得較早之版本，或者其他作業系統的版本，也可以自行選擇。另外，由於教材未必會隨 Python 官方網站介面以及新的環境版本之更改而更新，故實際操作步驟或畫面可能與上段敘述有所不同；請確定下載的是 Python 3 最新穩定版的 64 位元版本即可，否則可能會有些範例無法執行。

開始安裝，請確認上方紅圈處是否為 64-bit，以及下方紅圈處是否已全部勾選。若是，則可以點選 Install Now，以預設的路徑和內容安裝，安裝完成後按關閉即可。

安裝完成後，請呼叫出 command window（往後簡稱 cmd，中文為「命令提示字元」，點選 Windows 的「開始」，並在「搜尋程式及檔案」處輸入「cmd」即可呼叫），輸入「python」後，若看到以下畫面（可再次確認是否與紅圈處一致，除了版本編號可以更高以外），則代表安裝成功，可立即開始使用。

關於開發環境的取得，首先可至 https://www.python.org/點選「Downloads」→「Binaries」。 你在這裡可以發現 Python 有兩個版本，Python 2 和 Python 3，本課程將以 3 為主，但在某些狀況下，也會提到 2 與 3 的不同。 以下的安裝說明將以 Windows 為例，若使用其他作業系統，請參考網路上的相關資料。

開始安裝，請勾選最下方的全部選項，再點選 Install Now，以預設的路徑和內容安裝，安裝完成後按關閉即可。

安裝完成後，請呼叫出 command window (往後簡稱 cmd，中文為"命令提示字元"，點選 Windows 的"開始"，並在"搜尋程式及檔案"處輸入 cmd)， 輸入"python"(不含引號)後，若看到以下畫面，則代表安裝成功，即可開始使用。 另外，若已安裝過 Python 2，則需要將其中一個版本的 python.exe 修改名稱為 python2.exe 或 python3.exe，以免混淆。

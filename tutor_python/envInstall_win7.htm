本篇以 Windows 7 為例，說明 Python 開發環境的安裝；若使用其他作業系統，請參考網路上的相關資料。關於開發環境的取得，首先可至 https://www.python.org/ 點選「Downloads」→「Windows」→「Latest Python 3 Release ...」，並下載「Windows x86-64 executable installer」。事實上，你在這些步驟中，可以發現 Python 有兩個版本，Python 2 和 Python 3，本教材將以 Python 3 為主，但在某些狀況下，也會提到 Python 2 與 Python 3 的不同。另外，由於教材未必會隨 Python 官方網站介面之更改而更新，故實際操作步驟可能與上段敘述有所不同；請確定下載的是 Python 3 最新穩定版的 64 位元版本即可，否則可能會有些範例無法執行。

開始安裝，請確認紅圈處是否與圖片一致（除了版本編號可以更高以外），並勾選最下方的全部選項後，點選 Install Now，以預設的路徑和內容安裝，安裝完成後按關閉即可。

安裝完成後，請呼叫出 command window（往後簡稱 cmd，中文為「命令提示字元」，點選 Windows 的「開始」，並在「搜尋程式及檔案」處輸入「cmd」即可呼叫），輸入「python」後，若看到以下畫面（可再次確認是否與紅圈處一致，除了版本編號可以更高以外），則代表安裝成功，可立即開始使用。另外，若已安裝過 Python 2，則可能需要自行將其中一個版本的 python.exe，依據其版本修改名稱為 python2.exe 或 python3.exe，以免混淆。
